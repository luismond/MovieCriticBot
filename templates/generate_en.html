About Movie Critic Bot

The app

Movie Critic Bot can do two things:

  • Predict the sentiment of movie reviews
  • Generate movie reviews

Sentiment prediction

The prediction function implements a scikit-learn classifier.

Text generation

The generation function is an implementation of textgenrnn, a Python library for Natural Language Generation

The data

This bot was trained with 5,000 negative and positive reviews from IMDb.
